### Installation:
1. Clone this repository

2. To install dependencies:
```bash
npm install
```
3. To start the program:
```bash
node index.js
```
4. To run the tests:
```bash
npm test
```
5. After entering all the required information, the generated `HTML` will be in the `dist` directory named <ins>generatedTeam.html
